To Run The code for the first time 

1- in CMD or in an empty folder in VS code open the terminal 
2- write command git clone ....
3- write command python -m venv venv
4- write command venv\Scripts\activate
5- write command pip install -r requirements.txt
6- write command python -m app.main
you should be able to open http://127.0.0.1:5000 in the browser
